
## Description

This is the products-service, a microservice that has an API for managing the products data. 

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

You can see the result, and the documentation over [here](http://localhost:3000/api/#/), on a Swagger.

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
